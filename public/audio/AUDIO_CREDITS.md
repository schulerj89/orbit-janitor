# Audio Credits

The game does not require these files to run. If an audio file is missing or fails to decode, `AudioManager` and `MusicDirector` fall back to procedural Web Audio.

No ElevenLabs API key is committed, referenced from browser code, or required at runtime. These MP3 files were generated offline and committed as static assets.
